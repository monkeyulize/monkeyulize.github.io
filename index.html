LivestockCloud

LivestockCloud is an animal weight tracking and management platform. The responsive interface allows users to quickly record weights for animals under their care. A user can easily determine the future weight of their animals based on their past weights.
The application allows users to add other users as 'observers' which gives them the ability to share their animals data with other users. Observers are notified when users add new weights.

  • Client: Texas A&M (Senior Project)
  • Date: Dec 2015
  • Service: Web Development

Tic-Tac-Toe

GitHub

The...cat got it? This tic-tac-toe game allows players to use custom images instead of the traditional X and O, although those are used by default. Images are stored in an AWS S3 bucket.
Games are initiated by using the game ID's and clients are connected via websockets routed through a Node.js server.
